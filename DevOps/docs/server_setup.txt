Server setup:

Install docker:
	sudo apt get install docker

copy builder.py  and Dockerfile_build from DevOps builder directory to server home
copy listener.py, Dockerfile from DevOps/listener directory to server home
run docker build on Dockerfile_build to name image “build”
run "docker build -t listen ." in server home
run "docker build -t build -f ./Dockerfile_build ." in server home
